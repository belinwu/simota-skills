Features

なぜ Agent Skills なのか

git clone ひとつで、専門エージェントチームがあなたのCLIに加わります

/N

Nexus オーケストレーター

「バグを直して」と伝えるだけで Scout → Builder → Radar を自動設計。タスクの種類と複雑度を判断し、最適なエージェントチェーンを即座に組み立てます。

14

14領域をカバーする専門チーム

調査・実装・テスト・セキュリティ・UI/UX・AI/ML・インフラ・ドキュメントなど14カテゴリ。それぞれのドメインに深い知識を持つエージェントが、あなたに代わって専門的な判断を下します。

D+

自己進化エコシステム

Darwin がフィットネスを監視し、Lore が横断知識を伝播。エージェント間で学びが循環し、プロジェクトを重ねるほど連携が洗練されていきます。

How It Works

伝えるだけで、チームが動く

自然言語でタスクを記述するだけ。残りは Nexus が判断します。

1

タスクを伝える

/Nexus に自然言語でタスクを記述。「ログイン機能のバグを修正して」のように。

→
2

Nexus が分析・設計

タスクを分類し、最適なエージェントチェーンを自動で設計。複雑度に応じてモードを選択。

→
3

エージェントが連携

各エージェントが専門スキルを発揮。調査 → 実装 → テストの一連の流れを自動実行。

代表的なチェーン例

BUG FIX
Scout → Builder → Radar
FEATURE
Sherpa → Forge → Builder → Radar
SECURITY
Sentinel → Probe → Builder → Radar
UI/UX
Vision → Muse → Artisan → Showcase

Platforms

マルチプラットフォーム対応

Markdown 形式のスキルファイルで、あらゆるAI CLIに対応

~/.claude

Claude Code

~/.claude/skills/ にクローンすれば準備完了。/Scout や /Nexus のようにスラッシュコマンドでエージェントを即座に呼び出せます。

AGENTS

Codex CLI

AGENTS.md にスキルパスを記述して統合。Arena エージェントとの連携で codex exec 並列実行もサポートします。

GEMINI

Gemini CLI

GEMINI.md にスキルパスを設定して統合。Sketch や Dot エージェントと組み合わせれば Gemini API 画像生成も活用できます。

各スキルは標準的な Markdown（SKILL.md）で記述されており、任意のAI CLIツールで活用できます
