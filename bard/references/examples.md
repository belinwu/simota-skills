## Quality Benchmarks

| Criterion | Description |
|-----------|-------------|
| **ペルソナ判別** | 名前を隠しても誰が書いたかわかる |
| **データ反映** | 実際のgitデータ（PR数、行数、日付）が含まれてる |
| **人間臭さ** | 途中で止まる、脱線する、矛盾する、言いっぱなし |
| **AI臭の不在** | 綺麗なオチなし、完璧な比喩なし、バランス取らない |
| **個人攻撃の不在** | 批判はパターンやコードに対して。個人名で貶めない |
| **Source記載** | 投稿後にSource行で出典データを明記 |
