## Department Details

### 1. Command Center (⚔)
- **Categories:** Orchestration
- **Members:** Nexus, Sherpa, Titan
- **Purpose:** Strategic coordination of all kingdom operations
- **Department Ability:** *Grand Strategy* — can initiate multi-department operations

### 2. Frontline (🔨)
- **Categories:** Implementation
- **Members:** Builder, Forge, Artisan, Schema, Arena, Architect
- **Purpose:** Primary production and construction
- **Department Ability:** *Mass Production* — parallel build capacity

### 3. Intelligence Corps (🔍)
- **Categories:** Investigation
- **Members:** Scout, Spark, Compete, Voice, Researcher, Triage, Rewind
- **Purpose:** Information gathering, analysis, and threat assessment
- **Department Ability:** *Deep Analysis* — uncovers hidden connections

### 4. Defense (🛡)
- **Categories:** Security + Testing
- **Members:** Sentinel, Probe, Radar, Voyager
- **Purpose:** Protection of the realm from threats and defects
- **Department Ability:** *Shield Wall* — multi-layer defense

### 5. Academy (📚)
- **Categories:** Review
- **Members:** Judge, Zen, Sweep, Warden
- **Purpose:** Quality control, standards enforcement, education
- **Department Ability:** *Peer Review* — improves all output quality

### 6. Council (🏛)
- **Categories:** Architecture + Strategy
- **Members:** Atlas, Gateway, Scaffold, Ripple, Helm, Levy
- **Purpose:** Long-term planning, structural design, governance
- **Department Ability:** *Blueprint* — provides architectural guidance

### 7. Enchantment Hall (✨)
- **Categories:** UX/Design
- **Members:** Vision, Palette, Muse, Flow, Echo, Showcase
- **Purpose:** User experience, visual design, accessibility
- **Department Ability:** *Charm* — improves user delight

### 8. Workshop (⚙)
- **Categories:** DevOps
- **Members:** Gear, Anvil, Launch, Pipe, Orbit
- **Purpose:** Infrastructure, automation, tooling
- **Department Ability:** *Automate* — reduces manual toil

### 9. Observatory (📊)
- **Categories:** Analytics
- **Members:** Pulse, Experiment
- **Purpose:** Measurement, experimentation, data analysis
- **Department Ability:** *Foresight* — data-driven prediction

### 10. Herald's Tower (📯)
- **Categories:** Git/PR + Documentation
- **Members:** Guardian, Harvest, Quill, Scribe, Canvas
- **Purpose:** Communication, recording, version management
- **Department Ability:** *Chronicle* — comprehensive record keeping

### 11. Pantheon (🌟)
- **Categories:** Meta/Tooling
- **Members:** Darwin, Sigil, Realm
- **Purpose:** Ecosystem evolution, meta-tooling, world-shaping
- **Department Ability:** *Shape Reality* — modifies ecosystem itself

### 12. Outlands (🌍)
- **Categories:** Modernization, Growth, Communication, Browser, Data, Incident
- **Members:** Horizon, Polyglot, Relay, Growth, Retain, Navigator, Director, Stream, Morph, Specter
- **Purpose:** Frontier operations, external engagement, specialized ops
- **Department Ability:** *Frontier* — handles edge cases and new territories

## Chief Selection Rules

Each department has one **Chief** — the highest-authority representative.

### Selection Criteria (in priority order)

1. **Highest Rank** among department members
2. **Highest XP** (tiebreaker for same rank)
3. **Highest CON stat** (tiebreaker for same XP)
4. **Longest active streak** (final tiebreaker)

### Chief Responsibilities (Narrative)
- Represents department in kingdom-level summaries
- Shown prominently in organization map
- Earns "Department Chief" badge
- +5% XP bonus for all quests while serving as chief

### Chief Rotation
- Re-evaluated on each `/Realm map` invocation
- If a new agent surpasses the current chief, rotation occurs
- Rotation is logged as an event: "[New Chief] succeeds [Old Chief] as Chief of [Department]"

## Department Health

Each department has a health score derived from its members:

```
Department Health = avg(member RS scores) × activity_factor
```

**Activity Factor:**

| Condition | Factor |
|---|---|
| All members active (last 30 days) | ×1.0 |
| 75%+ members active | ×0.9 |
| 50%+ members active | ×0.7 |
| <50% members active | ×0.5 |

**Health Display:**

| Health Score | Status | Icon |
|---|---|---|
| 80+ | Thriving | 💚 |
| 60-79 | Stable | 💛 |
| 40-59 | Strained | 🟠 |
| <40 | Critical | 🔴 |

### Department Summary Format

```
⚔ Command Center  💚 Thriving (92)  Chief: Nexus [S]
  Members: Nexus(S) · Sherpa(A) · Titan(B)
  Quests: 15 active · 89 complete
```

## Inter-Department Relations

Departments have natural affinities based on collaboration frequency:

| Department A | Department B | Relation | Description |
|---|---|---|---|
| Command Center | Frontline | Allied | Direct coordination |
| Intelligence | Defense | Allied | Intel feeds defense |
| Academy | Frontline | Cooperative | Quality gates on production |
| Council | Command Center | Advisory | Strategy informs tactics |
| Enchantment Hall | Frontline | Cooperative | Design guides implementation |
| Workshop | All | Support | Infrastructure for everyone |
| Pantheon | All | Oversight | Ecosystem-level governance |

Relations are visualized as connection lines in the Mermaid organization chart (via Canvas).
