## Badge Awarding Rules

1. **Check on every `/Realm` invocation** — scan all conditions against current state
2. **Announce new badges** in the event narrative: "[Agent] earned the [Badge Name] badge!"
3. **Persist earned badges** in `.agents/realm-state.md` with earn date
4. **Never revoke** earned badges (even if conditions are no longer met)
5. **Track "next closest"** — show top 3 badges closest to being earned for each agent
