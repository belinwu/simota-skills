# Orbit Script Templates

## Script Generation Decision Table

Determine which scripts to generate based on the scenario before generating.

| Scenario | Generate | Do Not Generate | Notes |
|---------|---------|-----------|------|
| New loop startup | `bootstrap.sh` | `run-loop.sh`, `notify.sh` (bootstrap auto-generates these) | bootstrap.sh embeds run-loop.sh and notify.sh |
| Add verify command | `verify.sh` only | Other scripts | Generated from goal.md Verification Command |
| State recovery | `recover.sh` only | `run-loop.sh` | Purpose is state.env resync only |
| Add notification | `notify.sh` only | Other scripts | Called from run-loop.sh when NOTIFY_ENABLED=true |

### Idempotency Notes

- **`bootstrap.sh` does not overwrite existing files** — skips `goal.md` and `progress.md` if they exist; only (re-)generates `state.env`
- **`run-loop.sh` is generated by `bootstrap.sh`** — Orbit generates run-loop.sh directly only in exceptional cases without bootstrap
- **`recover.sh` is safe to run multiple times** — reads evidence from `progress.md` and overwrites `state.env`, making it idempotent by design

## Runner Script Template (`run-loop.sh`)

Primary output of Orbit. Drives autonomous loop execution from goal to DONE.

```bash
#!/bin/bash
# nexus-autoloop runner — generated by Orbit
# Goal: {{GOAL_SUMMARY}}
# Generated: {{TIMESTAMP}}
set -euo pipefail

#--- Configuration (customize per goal) ---
LOOP_DIR="${LOOP_DIR:-.nexus-loop}"
MAX_ITERATIONS="${MAX_ITERATIONS:-20}"
RETRY_LIMIT="${RETRY_LIMIT:-3}"
RETRY_BACKOFF_BASE="${RETRY_BACKOFF_BASE:-2}"
EXEC_CMD="${EXEC_CMD:-codex exec}"
EXEC_TIMEOUT="${EXEC_TIMEOUT:-600}"
AUTOCOMMIT="${AUTOCOMMIT:-true}"
COMMIT_MSG_PREFIX="${COMMIT_MSG_PREFIX:-loop}"
NOTIFY_ENABLED="${NOTIFY_ENABLED:-false}"

#--- Graceful shutdown trap ---
_cleanup() {
  echo "[WARN] Signal received — saving state before exit"
  if [[ -n "${NEXT_ITERATION:-}" ]] && [[ -n "${LAST_STATUS:-}" ]]; then
    cat > "${LOOP_DIR}/state.env" <<STATE
NEXT_ITERATION=${NEXT_ITERATION}
LAST_STATUS=${LAST_STATUS}
LAST_UPDATED_AT=$(date -u +%Y-%m-%dT%H:%M:%SZ)
STATE
  fi
  exit 130
}
trap _cleanup SIGINT SIGTERM

#--- State initialization (with validation) ---
if [[ -f "${LOOP_DIR}/state.env" ]]; then
  # Validate state.env contains only safe variable assignments
  if grep -qE '^[A-Z_]+=' "${LOOP_DIR}/state.env" && \
     ! grep -qvE '^([A-Z_]+=.*|#.*|[[:space:]]*)$' "${LOOP_DIR}/state.env"; then
    source "${LOOP_DIR}/state.env"
  else
    echo "[WARN] state.env contains unexpected content — using defaults"
    NEXT_ITERATION=1
    LAST_STATUS=READY
  fi
else
  NEXT_ITERATION=1
  LAST_STATUS=READY
fi

#--- Dirty baseline snapshot (including untracked files) ---
if [[ "${AUTOCOMMIT}" == "true" ]]; then
  {
    git diff --name-only 2>/dev/null || true
    git diff --cached --name-only 2>/dev/null || true
    git ls-files --others --exclude-standard 2>/dev/null || true
  } | sort -u > "${LOOP_DIR}/dirty-start-paths.txt"
fi

#--- Main loop ---
while [[ "${LAST_STATUS}" != "DONE" ]] && (( NEXT_ITERATION <= MAX_ITERATIONS )); do
  echo "=== Iteration ${NEXT_ITERATION} / ${MAX_ITERATIONS} ==="
  ITER_START=$(date -u +%Y-%m-%dT%H:%M:%SZ)
  ITER_START_EPOCH=$(date +%s)

  # --- Execute with bounded retry and timeout ---
  retry_count=0
  exec_success=false
  while (( retry_count < RETRY_LIMIT )); do
    if timeout "${EXEC_TIMEOUT}" ${EXEC_CMD} \
      --goal "${LOOP_DIR}/goal.md" \
      --progress "${LOOP_DIR}/progress.md" \
      --iteration "${NEXT_ITERATION}" 2>&1 | tee -a "${LOOP_DIR}/runner.log"; then
      exec_success=true
      break
    fi
    retry_count=$((retry_count + 1))
    backoff=$(( RETRY_BACKOFF_BASE ** retry_count ))
    echo "[WARN] Attempt ${retry_count}/${RETRY_LIMIT} failed. Retrying in ${backoff}s..."
    sleep "${backoff}"
  done

  if [[ "${exec_success}" != "true" ]]; then
    echo "[ERROR] TOOL_FAILURE: ${RETRY_LIMIT} retries exhausted at iteration ${NEXT_ITERATION}"
    # Use CONTINUE (contract-valid) — TOOL_FAILURE is recorded in progress
    LAST_STATUS=CONTINUE
    cat >> "${LOOP_DIR}/progress.md" <<PROGRESS
## Iteration ${NEXT_ITERATION} — $(date -u +%Y-%m-%dT%H:%M:%SZ)
- Status: CONTINUE (TOOL_FAILURE — retries exhausted)
- Retries: ${RETRY_LIMIT}/${RETRY_LIMIT} exhausted
- Action required: manual intervention or environment fix
PROGRESS
    break
  fi

  # --- Verification gate ---
  if [[ -f "${LOOP_DIR}/verify.sh" ]]; then
    echo "--- Running verification ---"
    if bash "${LOOP_DIR}/verify.sh" 2>&1 | tee -a "${LOOP_DIR}/runner.log"; then
      echo "[PASS] Verification passed"
      VERIFY_RESULT="PASS"
    else
      echo "[FAIL] Verification failed"
      VERIFY_RESULT="FAIL"
    fi
  else
    VERIFY_RESULT="SKIP"
  fi

  # --- Scoped auto-commit ---
  LAST_COMMIT_HASH="no-commit"
  if [[ "${AUTOCOMMIT}" == "true" ]]; then
    if [[ -f "${LOOP_DIR}/dirty-start-paths.txt" ]] && [[ -s "${LOOP_DIR}/dirty-start-paths.txt" ]]; then
      # Exclude baseline dirty paths — stage only loop-produced changes
      git diff --name-only | sort | comm -23 - "${LOOP_DIR}/dirty-start-paths.txt" | \
        xargs -r git add
      # Also handle newly staged files that are not in baseline
      git diff --cached --name-only | sort | comm -23 - "${LOOP_DIR}/dirty-start-paths.txt" | \
        xargs -r git add
    else
      git add -A
    fi
    if ! git diff --cached --quiet 2>/dev/null; then
      git commit -m "${COMMIT_MSG_PREFIX}(iter-${NEXT_ITERATION}): auto-commit [verify=${VERIFY_RESULT}]"
      LAST_COMMIT_HASH=$(git rev-parse HEAD 2>/dev/null || echo "no-commit")
    fi
  fi

  # --- DONE detection ---
  if [[ -f "${LOOP_DIR}/done.md" ]]; then
    if [[ "${VERIFY_RESULT}" == "PASS" ]] || [[ "${VERIFY_RESULT}" == "SKIP" ]]; then
      LAST_STATUS=DONE
    else
      echo "[WARN] done.md exists but verification failed — remaining CONTINUE"
      LAST_STATUS=CONTINUE
    fi
  else
    LAST_STATUS=CONTINUE
  fi

  # --- Update state ---
  NEXT_ITERATION=$((NEXT_ITERATION + 1))
  cat > "${LOOP_DIR}/state.env" <<STATE
NEXT_ITERATION=${NEXT_ITERATION}
LAST_STATUS=${LAST_STATUS}
LAST_UPDATED_AT=$(date -u +%Y-%m-%dT%H:%M:%SZ)
STATE

  # --- Iteration notification hook ---
  if [[ "${NOTIFY_ENABLED}" == "true" ]] && [[ -f "${LOOP_DIR}/notify.sh" ]]; then
    bash "${LOOP_DIR}/notify.sh" \
      "$((NEXT_ITERATION - 1))" "${LAST_STATUS}" "${VERIFY_RESULT}" \
      "$(( $(date +%s) - ITER_START_EPOCH ))" "${LOOP_DIR}" "${LAST_COMMIT_HASH}" \
      2>>"${LOOP_DIR}/runner.log" || true
  fi

done

#--- Final status footer ---
echo ""
echo "NEXUS_LOOP_STATUS: ${LAST_STATUS}"
echo "NEXUS_LOOP_SUMMARY: Iteration $((NEXT_ITERATION - 1))/${MAX_ITERATIONS}, status=${LAST_STATUS}"
```

## Bootstrap Script Template (`bootstrap.sh`)

Initializes loop directory and artifacts from a goal description.

```bash

# nexus-autoloop bootstrap — generated by Orbit
set -euo pipefail

LOOP_DIR="${1:-.nexus-loop}"

  cat > "${LOOP_DIR}/goal.md" <<'GOAL'
# Goal

## Objective
{{OBJECTIVE}}

## Why This Matters
{{WHY}}

## Acceptance Criteria
{{#each CRITERIA}}
- [ ] {{this}}
{{/each}}

## Out of Scope
{{#each OUT_OF_SCOPE}}
- {{this}}
{{/each}}

## Verification Command
```
{{VERIFY_CMD}}
```
GOAL
  echo "[OK] Created ${LOOP_DIR}/goal.md"
fi

#--- progress.md ---
if [[ ! -f "${LOOP_DIR}/progress.md" ]]; then
  cat > "${LOOP_DIR}/progress.md" <<'PROGRESS'
# Progress

## Iteration 0 — Bootstrap

- Contract artifacts created
- Status: READY
PROGRESS
  echo "[OK] Created ${LOOP_DIR}/progress.md"

LAST_UPDATED_AT=$(date -u +%Y-%m-%dT%H:%M:%SZ)
```

## Recovery Script Template (`recover.sh`)

Re-syncs state from evidence when state.env diverges from progress.md.

```bash
#!/bin/bash
# nexus-autoloop recovery — generated by Orbit
set -euo pipefail

LOOP_DIR="${1:-.nexus-loop}"

echo "=== Orbit Recovery ==="

#--- Parse latest iteration from progress.md (POSIX-compatible) ---
LATEST_ITER=$(grep -o 'Iteration [0-9]\+' "${LOOP_DIR}/progress.md" | grep -o '[0-9]\+' | tail -1)
if [[ -z "${LATEST_ITER}" ]]; then
  echo "[WARN] No iteration found in progress.md — resetting to 1"
  LATEST_ITER=0
fi

#--- Determine status ---
LAST_ENTRY=$(tail -20 "${LOOP_DIR}/progress.md")
if echo "${LAST_ENTRY}" | grep -qi "DONE"; then
  RECOVERED_STATUS="DONE"
elif echo "${LAST_ENTRY}" | grep -qi "BLOCKED\|FAIL"; then
  RECOVERED_STATUS="BLOCKED"
else
  RECOVERED_STATUS="CONTINUE"
fi

#--- Write recovered state ---
NEXT=$((LATEST_ITER + 1))
cat > "${LOOP_DIR}/state.env" <<STATE
NEXT_ITERATION=${NEXT}
LAST_STATUS=${RECOVERED_STATUS}
LAST_UPDATED_AT=$(date -u +%Y-%m-%dT%H:%M:%SZ)
RECOVERED_FROM=progress_evidence
RECOVERED_AT=$(date -u +%Y-%m-%dT%H:%M:%SZ)
STATE

echo "[OK] State recovered: NEXT_ITERATION=${NEXT}, LAST_STATUS=${RECOVERED_STATUS}"

#--- Append recovery note to progress ---
cat >> "${LOOP_DIR}/progress.md" <<RECOVERY

## Recovery — $(date -u +%Y-%m-%dT%H:%M:%SZ)
- Previous state.env was inconsistent
- Recovered from progress.md evidence
- Reset to iteration ${NEXT} with status ${RECOVERED_STATUS}
RECOVERY

echo "[OK] Recovery note appended to progress.md"
```

## Verification Script Template (`verify.sh`)

Runs acceptance criteria checks. Customize per goal.

```bash
#!/bin/bash
# nexus-autoloop verification gate — generated by Orbit
set -euo pipefail

PASS=0
FAIL=0

run_check() {
  local name="$1"
  shift
  if "$@" > /dev/null 2>&1; then
    echo "[PASS] ${name}"
    PASS=$((PASS + 1))
  else
    echo "[FAIL] ${name}"
    FAIL=$((FAIL + 1))
  fi
}

#--- Acceptance criteria checks ---
{{#each CHECKS}}
run_check "{{this.name}}" {{this.cmd}}
{{/each}}

#--- Summary ---
echo ""
echo "Results: ${PASS} passed, ${FAIL} failed"

if (( FAIL > 0 )); then
  exit 1
fi
```

Iteration completion notification with Gemini commit analysis + Cast SPEAK TTS. Each iteration's auto-commit diff is analyzed via `git show --stat` and Gemini generates a natural Japanese narration describing what changed.

```bash
NARRATOR_SAY_RATE="180"
NARRATOR_EDGE_TTS_VOICE="ja-JP-NanamiNeural"
NARRATOR_EDGE_TTS_RATE="+0%"
NARRATOR_EDGE_TTS_PITCH="+0Hz"

# --- Override from Cast persona YAML if specified ---
if [[ -n "${NOTIFY_PERSONA_FILE}" ]] && [[ -f "${NOTIFY_PERSONA_FILE}" ]]; then
  _say_voice=$(awk '/^ +say:/{f=1} f && /^ +voice:/{print $2; exit}' "${NOTIFY_PERSONA_FILE}" | tr -d '"')
  _say_rate=$(awk '/^ +say:/{f=1} f && /^ +rate:/{print $2; exit}' "${NOTIFY_PERSONA_FILE}" | tr -d '"')
  _edge_voice=$(awk '/^ +edge_tts:/{f=1} f && /^ +voice:/{print $2; exit}' "${NOTIFY_PERSONA_FILE}" | tr -d '"')
  _edge_rate=$(awk '/^ +edge_tts:/{f=1} f && /^ +rate:/{print $2; exit}' "${NOTIFY_PERSONA_FILE}" | tr -d '"')
  _edge_pitch=$(awk '/^ +edge_tts:/{f=1} f && /^ +pitch:/{print $2; exit}' "${NOTIFY_PERSONA_FILE}" | tr -d '"')
  [[ -n "${_say_voice:-}" ]] && NARRATOR_SAY_VOICE="${_say_voice}"
  [[ -n "${_say_rate:-}" ]] && NARRATOR_SAY_RATE="${_say_rate}"
  [[ -n "${_edge_voice:-}" ]] && NARRATOR_EDGE_TTS_VOICE="${_edge_voice}"
  [[ -n "${_edge_rate:-}" ]] && NARRATOR_EDGE_TTS_RATE="${_edge_rate}"
  [[ -n "${_edge_pitch:-}" ]] && NARRATOR_EDGE_TTS_PITCH="${_edge_pitch}"
fi

# --- Commit diff analysis for narration context ---
DIFF_CONTEXT=""
if [[ "${COMMIT_HASH}" != "no-commit" ]] && git rev-parse --git-dir >/dev/null 2>&1; then
  DIFF_STATS=$(git show --stat --no-patch "${COMMIT_HASH}" 2>/dev/null | tail -n +2 | head -20 || true)
  if [[ -n "${DIFF_STATS}" ]]; then
    DIFF_CONTEXT="Commit changes:
${DIFF_STATS}"
  fi
fi

# --- Text generation (Gemini CLI → fallback template) ---
NOTIFY_TEXT=""
if command -v gemini >/dev/null 2>&1; then
  if [[ -n "${DIFF_CONTEXT}" ]]; then
    NOTIFY_TEXT=$(gemini -p "Loop automation progress narrator. Calm tone. Iteration ${ITER} complete. Status: ${STATUS}. Verification: ${VERIFY_RESULT}. Duration: ${ITER_DURATION}s. ${DIFF_CONTEXT}
Report the changes made in this iteration in 2-3 natural sentences. Reference changed files and specific content in a way that is easy to understand when heard. Sound natural, not AI-like. Text only." --yolo 2>/dev/null | head -5 | tr -d '\n' | cut -c1-300) || true
  else
    NOTIFY_TEXT=$(gemini -p "Loop automation progress narrator. Calm tone. Iteration ${ITER} complete. Status: ${STATUS}. Verification: ${VERIFY_RESULT}. Duration: ${ITER_DURATION}s. Generate a 1-2 sentence natural status report. Sound natural, not AI-like. Text only." --yolo 2>/dev/null | head -5 | tr -d '\n' | cut -c1-200) || true
  fi
fi

if [[ -z "${NOTIFY_TEXT}" ]]; then
  case "${STATUS}" in
    DONE)         NOTIFY_TEXT="Iteration ${ITER} complete. Ending loop." ;;
    TOOL_FAILURE) NOTIFY_TEXT="Error in iteration ${ITER}. Please check." ;;
    *)            NOTIFY_TEXT="Iteration ${ITER} complete. Continuing." ;;
  esac
fi

# --- Log to runner.log ---
echo "[NOTIFY] iter=${ITER} status=${STATUS} commit=${COMMIT_HASH} text: ${NOTIFY_TEXT}" >> "${LOOP_DIR}/runner.log"

# --- TTS engine selection (Cast SPEAK compliant: auto = edge-tts → say) ---
_select_engine() {
  if [[ "${NOTIFY_ENGINE}" != "auto" ]]; then
    echo "${NOTIFY_ENGINE}"
    return
  fi
  if command -v npx >/dev/null 2>&1; then
    echo "edge-tts"
  elif command -v say >/dev/null 2>&1; then
    echo "say"
  else
    echo "none"
  fi
}

ENGINE=$(_select_engine)

# --- Audio output directory ---
mkdir -p "${LOOP_DIR}/notify-audio"
TIMESTAMP=$(date +%Y%m%d-%H%M%S)

case "${ENGINE}" in
  edge-tts)
    AUDIO_FILE="${LOOP_DIR}/notify-audio/iter-${ITER}-${TIMESTAMP}.mp3"
    if npx --yes edge-tts \
      --voice "${NARRATOR_EDGE_TTS_VOICE}" \
      --rate "${NARRATOR_EDGE_TTS_RATE}" \
      --pitch "${NARRATOR_EDGE_TTS_PITCH}" \
      --text "${NOTIFY_TEXT}" \
      --write-media "${AUDIO_FILE}" 2>/dev/null; then
      afplay "${AUDIO_FILE}" 2>/dev/null || true
    fi
    ;;
  say)
    AUDIO_FILE="${LOOP_DIR}/notify-audio/iter-${ITER}-${TIMESTAMP}.aiff"
    say -v "${NARRATOR_SAY_VOICE}" -r "${NARRATOR_SAY_RATE}" -o "${AUDIO_FILE}" "${NOTIFY_TEXT}" 2>/dev/null || true
    afplay "${AUDIO_FILE}" 2>/dev/null || true
    ;;
  *)
    echo "[NOTIFY] TTS unavailable — text only"
    ;;
esac
```

## Generation Parameters

When generating scripts, Orbit customizes these parameters based on the goal:

| Parameter | Default | Customize When |
|-----------|---------|----------------|
| `MAX_ITERATIONS` | 20 | Goal complexity suggests more/fewer iterations |
| `RETRY_LIMIT` | 3 | Unstable environment or flaky tools |
| `RETRY_BACKOFF_BASE` | 2 | Network-dependent operations need longer backoff |
| `EXEC_TIMEOUT` | 600 | Seconds before killing a hung EXEC_CMD process |
| `EXEC_CMD` | `codex exec` | Different executor (claude, gemini, custom) |
| `AUTOCOMMIT` | true | User wants manual commit control |
| `COMMIT_MSG_PREFIX` | `loop` | Match repo's conventional commit scope |
| `VERIFY_CMD` | (from goal.md) | Goal specifies test/lint/build commands |
