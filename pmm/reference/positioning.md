## What Positioning Is (and Is Not)

Positioning is the *context you set* so a segment instantly understands what the product is, who it is for, and why it is the best choice for them. It is a strategic decision, not a tagline (that is Prose) and not a story (that is Saga).

- **Is**: the market frame — target, category, unique value, differentiator.
- **Is not**: a slogan, a feature list, a narrative, or a competitive battle card (→ Compete).

One positioning statement per **target segment**. Different segments value different things; a single universal position is a non-position.

## Positioning Statement Template

The classic Moore "Crossing the Chasm" form — fill every slot, none generic:

```
For [target segment]
who [need / job-to-be-done],
[product] is a [market category]
that [key value / benefit].
Unlike [primary alternative],
[product] [primary differentiator — grounded in a shipped capability].
```

Rules:
- `target segment` must be nameable and reachable (tie to a Cast/Field persona).
- `key value` is an outcome, never a feature.
- `primary alternative` and `differentiator` come from Compete's frame — never invented.
- the differentiator must trace to a **shipped** capability with evidence; if it is roadmap, it is not yet a differentiator.

## Dunford Components (deliberate positioning)

When the Moore template feels forced, position from these building blocks (April Dunford, *Obviously Awesome*) and assemble the statement after:

1. **Competitive alternatives** — what would the customer use if you didn't exist? (Compete input.)
2. **Unique attributes** — what do you have that alternatives don't? (Shipped capabilities, evidence-grounded.)
3. **Value** — what those attributes *do for the customer* (outcome, not feature).
4. **Target customers** — who cares most about that value (the beachhead segment).
5. **Market category** — the frame of reference that makes the value obvious.

Work right-to-left from value: capabilities → value → who values it most → category that frames it.

## Category Choice

The category sets the customer's expectations and comparison set. Three moves:
- **Play in an existing category** — easy to understand, crowded comparison. Default for most.
- **Subsegment an existing category** — "X for Y" (e.g., "CRM for solo founders"). Lower risk than net-new.
- **Design a new category** — highest cost/reward; only when the value genuinely has no existing frame. Category design is a Helm/Compete-heavy strategic bet — flag it, do not declare a new category casually.

State the category explicitly; an unstated category lets the market pick the wrong comparison.

## Positioning Map

A 2-axis visual placing the product against alternatives. PMM **consumes Compete's competitive data** to plot it — it does not re-research competitors.

```
Pick two axes that matter to the segment (NOT two you happen to win on):
  e.g. y = "depth of compliance evidence", x = "time-to-setup"
Plot: own product + each alternative (from Compete) at honest positions.
Read: find the open quadrant the segment values and you can credibly own.
```

Hand the plotted map to Canvas for rendering (`PMM_TO_CANVAS_HANDOFF`). Axes must be customer-relevant, not vanity axes the product trivially tops.

## Anti-Patterns

| Anti-pattern | Why it fails | Fix |
|--------------|--------------|-----|
| "For everyone" | persuades no one; no segment owns it | name one beachhead segment |
| Feature-as-position | features aren't a reason to care | translate to value/outcome |
| Self-declared "best/leading" | no competitive frame, not credible | anchor to a Compete alternative |
| Vaporware differentiator | roadmap line dressed as advantage | require shipped-capability evidence |
| Vanity axes on the map | wins a comparison nobody values | pick axes the segment ranks on |

## Confidence & Grounding

| Confidence | Condition |
|-----------|-----------|
| High | Differentiator is a shipped capability (evidence) AND competitive frame is from Compete AND segment is a real persona |
| Medium | One side inferred (e.g. capability shipped but competitive frame is assumed, or persona provisional) |
| Low | Differentiator unverified, or no competitive frame, or segment undefined — usually means GROUND is incomplete; broaden before shipping the statement |

Downgrade and state the reason. A confident-sounding position built on an unverified differentiator is the most expensive PMM error — it sends sales to make a promise the product can't keep.
