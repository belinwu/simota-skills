# 記帳パターンリファレンス

## 記帳方式の比較

| 項目 | 複式簿記 | 簡易簿記 |
|------|---------|---------|
| 青色申告特別控除 | 55万円 or 65万円 | 10万円 |
| 必要帳簿 | 仕訳帳・総勘定元帳 | 現金出納帳・売掛帳等 |
| 決算書 | 貸借対照表+損益計算書 | 損益計算書のみ |
| 難易度 | 高（会計ソフト推奨） | 低 |
| 推奨対象 | 事業所得がある全個人事業主 | 所得が少額の場合 |

## 複式簿記の基本

### 勘定科目一覧（個人事業主向け）

#### 資産

| 科目 | 内容 | 例 |
|------|------|---|
| 現金 | 手持ち現金 | 事業用現金 |
| 普通預金 | 事業用口座 | 屋号口座 |
| 売掛金 | 未回収の売上代金 | 請求済み・未入金 |
| 前払費用 | 支払済み・未享受のサービス | 年払い保険料 |
| 工具器具備品 | 10万円以上の備品 | PC、デスク |
| 車両運搬具 | 事業用車両 | 営業車 |
| 事業主貸 | 事業→個人への資金移動 | 生活費引出し |

#### 負債

| 科目 | 内容 | 例 |
|------|------|---|
| 買掛金 | 未払いの仕入代金 | — |
| 未払金 | 未払いの経費 | クレカ未払分 |
| 預り金 | 一時的に預かった金銭 | 源泉徴収税 |
| 借入金 | 借入金 | 事業用ローン |
| 事業主借 | 個人→事業への資金移動 | 自己資金投入 |

#### 収益

| 科目 | 内容 |
|------|------|
| 売上高 | 本業の収入 |
| 雑収入 | 本業以外の収入（利息等） |

#### 費用（必要経費）

| 科目 | 内容 | 家事按分の可能性 |
|------|------|-----------------|
| 租税公課 | 固定資産税、自動車税、印紙税 | △（事業用資産分） |
| 荷造運賃 | 配送料、梱包材 | × |
| 水道光熱費 | 電気・ガス・水道 | ○（在宅勤務） |
| 旅費交通費 | 電車、タクシー、出張 | × |
| 通信費 | 電話、インターネット、郵便 | ○（在宅勤務） |
| 広告宣伝費 | 広告、PR | × |
| 接待交際費 | 取引先との飲食、贈答 | × |
| 損害保険料 | 事業用保険 | △ |
| 修繕費 | 事務所・設備の修理 | △ |
| 消耗品費 | 10万円未満の物品 | △ |

| 福利厚生費 | — | ×（個人事業主本人は不可） |
| 給料賃金 | 従業員給与 | × |
| 外注工賃 | 外部委託費 | × |
| 地代家賃 | 事務所賃料 | ○（自宅兼用） |
| 雑費 | 上記に該当しないもの | — |
| 専従者給与 | 青色事業専従者への給与 | ×（届出必要） |

## 家事按分

事業用と私用の両方に使う費用を合理的な基準で按分する。

### 按分基準の例

| 経費 | 按分基準 | 計算方法 |
|------|---------|---------|
| 地代家賃 | 面積比 | 事業用面積 ÷ 総面積 |
| 水道光熱費 | 使用時間 or 面積比 | 事業使用時間 ÷ 総時間 |
| 通信費 | 使用割合 | 事業用通話・通信 ÷ 総量 |
| 車両関連費 | 走行距離比 | 事業走行距離 ÷ 総走行距離 |
| PC関連費 | 使用時間比 | 事業使用時間 ÷ 総使用時間 |

### 按分率の目安（フリーランス・在宅勤務）

| 経費 | 一般的な按分率 | 根拠 |
|------|-------------|------|
| 家賃 | 30〜50% | 作業部屋の面積比 |
| 電気代 | 30〜50% | 使用時間比 |
| インターネット | 50〜80% | 事業利用割合 |
| 携帯電話 | 50〜70% | 事業通話割合 |

> **注意**: 按分率は実態に基づく合理的な基準が必要。根拠を記録しておくこと。

## 減価償却

### 方法

| 方法 | 計算 | 個人事業主の原則 |
|------|------|----------------|
| 定額法 | 取得価額 × 定額法償却率 | **原則（届出不要）** |
| 定率法 | 未償却残高 × 定率法償却率 | 届出が必要 |

### 耐用年数の例

| 資産 | 耐用年数 | 定額法償却率 |
|------|---------|------------|
| パソコン | 4年 | 0.250 |
| 事務机・椅子 | 8年（金属）/5年（木製） | 0.125/0.200 |
| コピー機 | 5年 | 0.200 |
| 普通自動車 | 6年 | 0.167 |
| 軽自動車 | 4年 | 0.250 |
| 建物（木造） | 22年 | 0.046 |
| 建物（RC造） | 47年 | 0.022 |
| ソフトウェア（自社利用） | 5年 | 0.200 |

### 少額資産の特例

| 区分 | 取得価額 | 処理方法 | 要件 |
|------|---------|---------|------|
| 少額減価償却資産 | 10万円未満 | 全額経費（消耗品費） | 全事業者 |
| 一括償却資産 | 10万円以上20万円未満 | 3年均等償却 | 全事業者 |
| 青色申告の特例 | 10万円以上30万円未満 | 全額経費 | 青色申告者（年間300万円限度） |

### 年の途中で取得した場合

```
減価償却費 = 取得価額 × 償却率 × 使用月数/12
（月の途中で取得 → その月を1ヶ月としてカウント）
```

## よくある仕訳パターン

### 売上関連

```
【売上計上（請求書発行時）】
  売掛金 100,000 / 売上高 100,000

【入金時】
  普通預金 89,790 / 売掛金 100,000
  事業主貸 10,210     （源泉徴収税 10.21%）
```

### 経費関連

```
【事業用品を現金で購入】
  消耗品費 5,000 / 現金 5,000

【家賃を銀行振込（按分率40%）】
  地代家賃 40,000 / 普通預金 100,000
  事業主貸 60,000

【クレジットカード利用】
  通信費 3,000 / 未払金 3,000  ← 利用時
  未払金 3,000 / 普通預金 3,000  ← 引落時

【個人の口座から事業経費を支払い】
  消耗品費 10,000 / 事業主借 10,000
```

### 決算仕訳

```
【減価償却（定額法）】
  減価償却費 62,500 / 工具器具備品 62,500
  （PC 250,000円、4年定額法: 250,000 × 0.250）

【家事按分の決算整理】
  事業主貸 72,000 / 水道光熱費 72,000
  （年間120,000円、事業按分率40%、個人分60%=72,000円を振替）
```

## 帳簿の保存期間

| 書類 | 保存期間 |
|------|---------|
| 帳簿（仕訳帳・総勘定元帳） | 7年 |
| 決算書類（貸借対照表・損益計算書） | 7年 |
| 請求書・領収書（取引金額の証拠） | 7年（前々年の所得300万円以下は5年） |
| その他の書類（見積書等） | 5年 |

### 電子帳簿保存法

- 2024年1月1日以降、電子取引データの電子保存が義務化
- メールで受領した請求書・領収書はデータのまま保存（印刷して紙保存は不可）
- 検索要件: 取引年月日、取引金額、取引先で検索可能なこと

## 個人口座兼用の仕訳パターン

事業専用口座を持たず、個人口座で事業取引を行う場合は**事業主貸/事業主借**で処理する。

### 基本パターン

```
【個人口座に事業売上が入金】
  事業主貸 XXX,XXX / 売上高 XXX,XXX

【個人口座に売掛金が入金（源泉徴収あり）】
  事業主貸 XXX,XXX / 売掛金 XXX,XXX
  事業主貸  XX,XXX     （源泉徴収税額分）

【個人口座・個人クレカで事業経費を支払い】
  通信費 X,XXX / 事業主借 X,XXX

【個人口座から家賃引落し（按分あり）】
  地代家賃 XX,XXX / 事業主借 XXX,XXX
  事業主貸 XX,XXX      （個人負担分）
```

> **ポイント**: 事業主貸/事業主借は損益に影響しない資本の勘定科目。年間の累計額を貸借対照表に記載する。

## 年末売掛金の処理（発生主義）

所得税法第36条に基づき、収入は**役務提供日（権利確定日）**が属する年度で計上する。

```
【12月に役務提供、翌年1月に入金】
  (当年) 売掛金 XXX,XXX / 売上高 XXX,XXX  ← 当年の売上
  (翌年) 普通預金 XXX,XXX / 売掛金 XXX,XXX  ← 翌年は売掛金の回収

【前年12月に提供、当年1月に入金】
  当年の売上には含めない（前年の売掛金回収として処理）
```

> **注意**: 入金日ではなく役務提供日で年度を判定する。年度をまたぐ取引は特に注意。

## IT/ソフトウェア開発の経費科目

IT/ソフトウェア開発に特有の経費は、e-Taxの標準科目にない場合がある。任意科目として追加する。

| 支出内容 | 推奨勘定科目 | e-Tax区分 |
|---------|------------|----------|
| SaaS利用料（GitHub, AWS, Figma等） | **支払手数料** | 任意科目 |
| 技術書籍・電子書籍 | **新聞図書費** | 任意科目 |
| カンファレンス参加費・オンライン講座 | **研修費** | 任意科目 |
| コワーキングスペース月額 | 地代家賃 | 標準科目 |
| ドメイン・サーバー費用 | 支払手数料 or 通信費 | 任意/標準 |
| 外注費（デザイン等） | 外注工賃 | 標準科目 |

→ 詳細: `references/salary-plus-side-business.md` IT/ソフトウェア開発の経費分類

## 貸借対照表の固定資産

貸借対照表に記載する固定資産は**帳簿価額（純額）**で記載する。

```
帳簿価額 = 取得価額 − 減価償却累計額

例: PC 250,000円、2年目末（定額法・4年）
  減価償却累計額 = 250,000 × 0.250 × 2 = 125,000
  帳簿価額 = 250,000 − 125,000 = 125,000 ← この金額を記載
```

> 取得価額（原価）ではなく、帳簿価額（純額）を記載すること。

## 端数調整パターン

e-Taxと会計ソフトの間で端数処理方式（切捨て/四捨五入/切上げ）が異なり、1円〜数円のズレが発生することがある。

| ズレ幅 | 対処法 |
|--------|--------|
| 1円〜数円 | 事業主貸/事業主借で調整 |
| それ以上 | 仕訳の漏れ・二重計上を確認 |

```
【貸借対照表で資産側が1円多い場合】
  事業主貸 1 / [差異のある科目] 1

【貸借対照表で負債+資本側が1円多い場合】
  [差異のある科目] 1 / 事業主借 1
```

> **原則**: e-Taxの計算結果を正とし、会計ソフト側を合わせる。
